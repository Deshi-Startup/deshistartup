# Deshi Startup (দেশে স্টার্টআপ)

[![Star on GitHub](https://img.shields.io/github/stars/Deshi-Startup/deshistartup.svg?style=social)](https://github.com/Deshi-Startup/deshistartup)

An open-source, practical playbook in Bangla on how to start, build, grow, and scale a startup in Bangladesh. 

Our goal is to become the best resource for startups specifically in Bangladesh, so that anyone—even an SSC/HSC student—can follow along and build their own startup.

## 🤝 How to Contribute

This is a community-driven project and we welcome contributions from founders, lawyers, accountants, and developers!

Whether it's fixing a typo, updating a tax law, or writing a whole new guide—your help makes this better for everyone.

### Getting Started:
1. **Fork** the repository.
2. **Clone** your fork locally.
3. Install dependencies: `npm install`
4. Run the development server: `npm run dev`
5. Make your changes in the `app/` directory (it uses Nextra with MDX).
6. **Submit a Pull Request (PR)**!
