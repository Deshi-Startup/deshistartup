# ট্রেড লাইসেন্স (Trade License)

> **সারকথা:** বাংলাদেশে যেকোনো ব্যবসা চালাতে হলে ট্রেড লাইসেন্স বাধ্যতামূলক। এটি আপনার স্থানীয় সিটি কর্পোরেশন বা পৌরসভা থেকে নিতে হয়। ট্রেড লাইসেন্স ছাড়া ব্যবসা চালানো অবৈধ এবং জরিমানা হতে পারে।

---

## ট্রেড লাইসেন্স কী?

ট্রেড লাইসেন্স হলো একটি সরকারি অনুমতিপত্র যা প্রমাণ করে যে আপনার ব্যবসা স্থানীয় সরকারের নিয়ম মেনে চালানোর অনুমতি পেয়েছে।

**কেন দরকার:**
- আইনত ব্যবসা চালানোর জন্য বাধ্যতামূলক
- ব্যাংক অ্যাকাউন্ট খুলতে লাগে
- Supplier/Partner-দের বিশ্বাসযোগ্যতা দেখাতে
- VAT/BIN রেজিস্ট্রেশনে লাগতে পারে
- দোকান/অফিসের চুক্তিতে লাগে

---

## কোথায় আবেদন করবেন?

আপনার ব্যবসার অবস্থান অনুযায়ী:

| এলাকা | কোথায় আবেদন করবেন |
|-------|-------------------|
| ঢাকা উত্তর | [DNCC](https://dncc.gov.bd) — ঢাকা উত্তর সিটি কর্পোরেশন |
| ঢাকা দক্ষিণ | [DSCC](https://dscc.gov.bd) — ঢাকা দক্ষিণ সিটি কর্পোরেশন |
| চট্টগ্রাম | [CCC](https://ccc.gov.bd) — চট্টগ্রাম সিটি কর্পোরেশন |
| সিলেট | [SCC](https://scc.gov.bd) — সিলেট সিটি কর্পোরেশন |
| রাজশাহী | [Rajshahi CC](https://rajshahi.gov.bd) — রাজশাহী সিটি কর্পোরেশন |
| খুলনা | [KCC](https://kcc.gov.bd) — খুলনা সিটি কর্পোরেশন |
| অন্যান্য পৌর এলাকা | সংশ্লিষ্ট পৌরসভা/মিউনিসিপ্যালিটি |

## আবেদনের জন্য কী কী লাগবে?

### প্রয়োজনীয় কাগজপত্র

| কাগজ | বিবরণ |
|------|--------|
| **আবেদন ফর্ম** | সংশ্লিষ্ট সিটি কর্পোরেশন/পৌরসভার ফর্ম |
| **জাতীয় পরিচয়পত্র (NID)** | আবেদনকারীর NID-এর কপি |
| **পাসপোর্ট সাইজ ছবি** | ১-২ কপি |
| **টিআইএন সার্টিফিকট (TIN)** | Tax Identification Number |
| **ভাড়ার চুক্তিনামা** | দোকান/অফিস ভাড়ার ক্ষেত্রে |
| **নকশা/ম্যাপ** | দোকান/অফিসের layout (কিছু জায়গায়) |
| **ইউটিলিটি বিল** | গ্যাস/পানি/বিদ্যুৎ বিল (ঠিকানা প্রমাণ) |
| **পূর্ববর্তী ট্রেড লাইসেন্স** | নবায়নের ক্ষেত্রে |

**কোম্পানির ক্ষেত্রে অতিরিক্ত:**
- Certificate of Incorporation
- MoA (Memorandum of Association)
- Board Resolution (ট্রেড লাইসেন্সের জন্য)

---

## আবেদন প্রক্রিয়া — স্টেপ বাই স্টেপ

### ঢাকা সিটি কর্পোরেশনের উদাহরণ

**অফলাইন (সরাসরি গিয়ে):**

1. **সিটি কর্পোরেশন অফিসে যান** — আপনার ওয়ার্ডের অফিসে
2. **ফর্ম সংগ্রহ করুন** — ট্রেড লাইসেন্স আবেদন ফর্ম
3. **ফর্ম পূরণ করুন** — সব তথ্য সঠিকভাবে লিখুন
4. **কাগজপত্র সংযুক্ত করুন** — সব প্রয়োজনীয় কাগজ attach করুন
5. **ফি জমা দিন** — নির্ধারিত ফি পরিশোধ করুন
6. **পরিদর্শন** — সিটি কর্পোরেশনের কর্মকর্তা আপনার দোকান/অফিস পরিদর্শন করতে পারেন
7. **লাইসেন্স সংগ্রহ** — approval পেলে ট্রেড লাইসেন্স সংগ্রহ করুন

**অনলাইন (যেখানে available):**

1. সংশ্লিষ্ট সিটি কর্পোরেশনের ওয়েবসাইটে যান
2. "Trade License" সেকশনে যান
3. অনলাইন আবেদন ফর্ম পূরণ করুন
4. কাগজপত্র আপলোড করুন
5. ফি অনলাইনে পরিশোধ করুন
6. Status check করুন
7. Download/সংগ্রহ করুন

## খরচ (ফি)

ট্রেড লাইসেন্স ফি ব্যবসার ধরন ও আয় অনুযায়ী ভিন্ন:

| ব্যবসার ধরন | আনুমানিক বার্ষিক ফি |
|-------------|-------------------|
| ছোট দোকান (মাসিক আয় ৳২৫,০০০ পর্যন্ত) | ৳৫০০ - ৳২,০০০ |
| মাঝারি দোকান/অফিস | ৳২,০০০ - ৳৫,০০০ |
| বড় দোকান/শোরুম | ৳৫,০০০ - ৳১৫,০০০ |
| ই-কমার্স/অনলাইন ব্যবসা | ৳২,০০০ - ৳১০,০০০ |
| রেস্তোরাঁ/খাবারের দোকান | ৳৫,০০০ - ৳২০,০০০ |
| ফ্যাক্টরি/উৎপাদন | ৳১০,০০০ - ৳৫০,০০০+ |

> **দ্রষ্টব্য:** ফি সিটি কর্পোরেশন ও ওয়ার্ড অনুযায়ী ভিন্ন হতে পারে। সঠিক ফি জানতে সংশ্লিষ্ট অফিসে যোগাযোগ করুন।

## নবায়ন (Renewal)

ট্রেড লাইসেন্স **প্রতি বছর** নবায়ন করতে হয়।

| বিষয় | বিবরণ |
|-------|--------|
| **নবায়নের সময়** | সাধারণত জুন-জুলাই (প্রতি বছর) |
| **বিলম্ব ফি** | সময়মতো না করলে জরিমানা (৫০%-১০০% অতিরিক্ত) |
| **কীভাবে** | একই অফিসে/ওয়েবসাইটে আবেদন |
| **প্রয়োজনীয় কাগজ** | পূর্ববর্তী লাইসেন্স, TIN, ভাড়ার চুক্তি, ফি |

> **টিপস:** ক্যালেন্ডারে নবায়নের তারিখ মarked রাখুন। বিলম্ব করলে জরিমানা বাড়ে।

## বিশেষ ক্ষেত্র

### অনলাইন ব্যবসার ট্রেড লাইসেন্স

আপনার ব্যবসা যদি সম্পূর্ণ অনলাইনে হয় (কোনো physical দোকান নেই):

- আপনার **বাসার ঠিকানা** ব্যবহার করে ট্রেড লাইসেন্স নিতে পারেন
- কিছু সিটি কর্পোরেশন "Home-based business" বা "E-commerce" category তে আলাদা ফি structure আছে
- **Warehouse/Office** থাকলে সেই এলাকার সিটি কর্পোরেশন থেকে নিতে হবে

### একাধিক শাখা

প্রতিটি শাখার জন্য **আলাদা ট্রেড লাইসেন্স** লাগবে — প্রতিটি শাখার এলাকার সিটি কর্পোরেশন/পৌরসভা থেকে।

### খাদ্য ব্যবসা

খাবারের ব্যবসা (রেস্তোরাঁ, ক্যাটারিং, ফুড ডেলিভারি) হলে অতিরিক্ত লাগতে পারে:
- **Food Safety License** — BSTI / Food Safety Authority
- **Environment clearance** — পরিবেশ অধিদপ্তর (বড় প্রতিষ্ঠানের ক্ষেত্রে)

## সাধারণ ভুল

| ❌ ভুল | ✅ সঠিক |
|--------|---------|
| ট্রেড লাইসেন্স ছাড়াই ব্যবসা শুরু | আগে লাইসেন্স নিন, তারপর শুরু করুন |
| নবায়ন ভুলে যাওয়া | ক্যালেন্ডারে মার্ক করুন, সময়মতো নবায়ন করুন |
| ভুল ওয়ার্ডে আবেদন | আপনার ব্যবসার সঠিক এলাকার ওয়ার্ড অফিসে যান |
| ফি না দিয়ে চালানো | জরিমানা বেশি হবে, ব্যবসা বন্ধ হতে পারে |
| Personal ঠিকানা ব্যবহার না করা | অনলাইন ব্যবসায় বাসা/অফিসের ঠিকানা ব্যবহার করুন |
