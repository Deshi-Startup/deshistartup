# Directory – Verified Bangladesh Startup Ecosystem List

> **In short:** This directory is not a hand-maintained prose table. Entries come from `data/directory/*.json`, so the same data can later power filters, APIs, an AI assistant, or an MCP layer. To add an entry, edit the JSON with name, type, stage, sectors, application path, source, and `lastVerified`.

## How to add an entry

Verify the organization through an official website or reliable public profile before adding it. If cheque size, deadline, or equity terms are not public, do not guess; keep `Not publicly stated`. Re-verify the directory quarterly.
