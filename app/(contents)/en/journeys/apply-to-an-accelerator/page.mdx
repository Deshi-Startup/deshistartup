# I want to apply to Startup Bangladesh or an accelerator

> **TL;DR:** Some programmes provide grants, some make equity investments, and others offer training or mentoring only. Match the stage, terms, time commitment and present constraint before applying. This path runs from eligibility through the application and pitch.

This path is for founders who want to apply to Startup Bangladesh, the iDEA project, or an accelerator/incubator. A strong application is more than filling a form – it's presenting a clear problem, proof, and team.

## The path, step by step

1. **Understand Startup Bangladesh.** Check the state-backed venture capital company's current investment type and eligibility. → [Startup Bangladesh](/en/funding/startup-bangladesh)
2. **Check eligibility.** Match what due diligence requires up front. → [Startup Bangladesh due diligence checklist](/en/funding/startup-bangladesh-dd)
3. **Research programs.** Find the program that fits your stage. → [Government program application guide](/en/ecosystem/government-programs)
4. **Look at grants and competitions.** Pick relevant grants and competitions. → [Grants and competitions](/en/funding/grants-competitions)
5. **Prepare application assets and pitch.** Organize the pitch deck and core message. → [Pitch deck template](/en/funding/pitch-deck)
6. **Browse the accelerator directory.** Use the source and last-verification details. → [Accelerator directory](/en/directory/accelerators)
7. **Follow the event calendar.** Don't miss deadlines and events. → [Event & opportunity calendar](/en/ecosystem/event-calendar)

## Checklist for this path

- [ ] Matched eligibility and required documents
- [ ] Picked a suitable program/accelerator
- [ ] Prepared the pitch deck and application assets
- [ ] Put application deadlines on the calendar
- [ ] Planned follow-up and reporting
