# Journeys – step-by-step paths by goal

> **TL;DR:** You do not need every guide at once. Choose the present goal, complete the first task in that path, then use the result to move forward. If a linked page carries a **“to be written”** notice, it is still unwritten; continue the previous task or use the nearest completed guide.

Reading guides one by one is easy, but a single guide can't answer "what exactly should I do now, and what comes after that?" These paths do exactly that: they break a specific goal (like "start an online business" or "register a company") into a few ordered steps and point you to the most relevant guide for each.

Not sure which path fits you? Start with the [starter roadmap](/en/start-here) or the [founder path chooser](/en/start-here/founder-path-chooser).
